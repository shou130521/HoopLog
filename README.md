# HoopLog

## アプリ概要

HoopLogは、NBAファンが観戦記録を残し、推し選手を管理できるWebアプリです。

SNSでは試合の感想や推し選手の活躍がタイムラインに埋もれてしまい、後から振り返ることが難しいという課題を解決することを目的としています。

観戦記録と推し選手を一元管理することで、自分だけのNBA観戦ログとして活用できます。

将来的には、同じチームや選手を応援するファン同士が交流できるコミュニティ機能の実装を目指しています。

---

## 制作背景

私はNBA観戦が趣味で、試合を見た感想や印象に残ったプレー、推し選手の活躍をSNSやメモアプリに記録していました。

しかし、SNSでは投稿がすぐにタイムラインに埋もれてしまい、メモアプリでは試合や選手ごとに整理して振り返ることが難しいと感じていました。

そこで、観戦記録と推し選手をまとめて管理し、試合の思い出をいつでも振り返ることができるサービスがあれば便利だと考え、HoopLogを制作しました。

---

## MVP機能

- ユーザー登録・ログイン
- 観戦記録の投稿
- 観戦記録一覧表示
- 観戦記録詳細表示
- 観戦記録編集・削除
- 推し選手の登録・管理
- 観戦記録と推し選手の紐付け
