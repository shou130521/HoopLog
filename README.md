# HoopLog

## アプリ概要

HoopLogは、NBAファンが観戦記録を残し、推し選手を管理できるWebアプリです。

SNSでは流れてしまう観戦の思い出や推し選手の活躍を記録し、後から振り返ることを目的としています。

将来的には、同じチーム・同じ選手を応援するファン同士が交流できるサービスを目指しています。

---

## MVP機能

- ユーザー登録・ログイン
- 観戦記録の投稿
- 観戦記録一覧表示
- 観戦記録詳細表示
- 観戦記録編集・削除
- 推し選手の登録・管理
- 観戦記録と推し選手の紐付け
